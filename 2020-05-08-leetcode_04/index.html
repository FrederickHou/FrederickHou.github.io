LeetCode 链表两数相加

题目

给出两个 非空 的链表用来表示两个非负的整数。其中，它们各自的位数是按照 逆序 的方式存储的，并且它们的每个节点只能存储 一位 数字。
如果，我们将这两个数相加起来，则会返回一个新的链表来表示它们的和。
您可以假设除了数字 0 之外，这两个数都不会以 0 开头。

示例：

输入：(2 -> 4 -> 3) + (5 -> 6 -> 4)
输出：7 -> 0 -> 8
原因：342 + 465 = 807

解题方法

递归检测

思路

在做两位数相加时要注意进位问题，当前节点值保留上一次计算的进位值和当前两节点数相加所取的模的值。同事要考虑两个链表长度不同的case，和当长度相同时最后一位相加会产生进位的case。
